# Progetto Coreform

Sistema di orientamento universitario ufficiale per le Università Pegaso, Mercatorum e San Raffaele.

## Panoramica
Coreform è un servizio di orientamento e assistenza agli studi creato per supportare lo studente nella scelta del Corso di Laurea più adatto alle sue esigenze, nell'immatricolazione e avvio del percorso universitario.

## Stack Tecnologico
- Laravel (versione 10.x)
- PHP 8.1
- Bootstrap 5
- MySQL
- Server: NGINX su VPS
